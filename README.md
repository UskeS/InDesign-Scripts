# InDesign-Scripts
Scripts of ExtendScript for Adobe InDesign.  
Adobe InDesign用のスクリプトです。

## Getting Started
Install the script in "Scripts" folder in InDesign.  
See each script for details.  
スクリプトをInDesignのScriptsフォルダにインストールしてください。詳細は各スクリプトを参照のこと。
